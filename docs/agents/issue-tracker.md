# Issue Tracker

- 仓库：`HeDaas-Code/Mortis`
- 模板：默认（暂未自定义）
- 关联：`#<num>`
- 关闭：`Closes #<num>` / `Fixes #<num>`

## 命名约定

Mortis 采用 **v0/v1/v2 ... 路线图** 命名（如 `Neural-Engine` 项目）：

- `v0-issue-N`——baseline 阶段（vault + 主人格骨架）
- `v1-issue-N`——mortis 架构核心
- `v2-issue-N`——扩展能力（多模型/插件系统）
- ...
